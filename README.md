SIT305-Task4.1p Personal Event Planner App

This app allows users to manage personal events. Users can add, view, update, and delete events. The app uses Room Database for local data storage and Jetpack Navigation Component for switching between screens.

Users can add new events with title, category, location, and date

Update existing events

Delete events

Use Bottom Navigation to switch between screens

Toast messages are used to remind users of validation errors and deletion confirmation

Technologies Used: Android Studio, Java, XML, Room Database RecyclerView, Jetpack Navigation Component, Bottom Navigation View
